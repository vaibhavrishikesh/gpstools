# Competitive Research

Research date: **2026-06-14**. Stats mostly from AppBrain (Play Store badges are rounded).

## A) The app we were asked to check

**GPS Photo: With Location & Map**
Package: `com.gonext.gpsphotolocation` · Developer: **gonext / Galaxy studio apps**

| Metric | Value |
|---|---|
| Downloads | **21M+** (AppBrain) · 10M+ Play badge |
| Rating | **4.10★** (~99k ratings) |
| Category rank | **#2 in Travel & Local** |
| APK size | 30.43 MB |
| Latest version | 4.2.0 — updated **4 Dec 2025** |
| Live since | Dec 2018 |
| Min Android | 5.0+ (API 21) |
| Monetization | Free + ads (IAP for ad-removal likely) |

**Features**
- Stamp **lat/long + map location** onto photos
- **Weather** at capture time
- GPS camera, real-time capture with **timestamp + geotag**
- **Map view** — browse photos by location
- Album organizer (sort by date / place)
- Customizable location-stamp layouts

**Developer scale (Galaxy studio apps):** 154 apps, ~200M total installs, active since 2013. High-volume utility-app factory — not a focused single-product team. This is a *clone-able* quality bar, not an untouchable incumbent.

## B) Category leader (the real benchmark)

**GPS Map Camera : Geo Tagging**
Package: `com.gpsmapcamera.geotagginglocationonphoto`

| Metric | Value |
|---|---|
| Downloads | **130M** |
| Rating | **3.34★** (~350k ratings) ⚠️ low |
| APK size | 34.87 MB |
| Latest version | 1.9.13 — updated **16 Apr 2026** |
| Live since | Mar 2020 |
| Min Android | 6.0+ |

**Why it leads (per its own marketing site):** NavIC (India's GPS) support, AI coordinate correction, customizable **report templates (PDF proof)**. Positioned for **government contractors, construction, NGO field work, auditors** — the "photo-proof" use case, not casual travel.

**⭐ Key insight:** 130M downloads but only **3.34★** = enormous install base with poor satisfaction. The market is proven and the leader is beatable on quality/UX. This is the single most important finding.

## C) Other notable players
- **Conota: GPS Timestamp Camera** (`com.gps.survey.cam`) — built for civil engineers, surveyors, architects, construction.
- **Timemark** — "photo evidence infrastructure" for field ops (capture, organize, share verified job photos).
- **GeoTag Camera – GPS Map Stamp** (`com.geotag.gpscamera`).
- Many near-duplicate "GPS Map Camera" clones from multiple devs (very crowded namespace).
- iOS: GPS Map Camera, GPS Map Camera: Geotag Photos.
